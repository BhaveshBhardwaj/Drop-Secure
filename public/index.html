Security

Military-grade encryption,
explained simply.

Every file transfer uses two independent layers of encryption so that even if one is compromised, your data stays safe.

01
ECDH Key Exchange

Both browsers generate ephemeral P-256 key pairs. Public keys are exchanged. Neither party ever sends their private key.

02
Shared Secret via HKDF

The ECDH shared secret is processed through HKDF-SHA-256 to derive a 256-bit AES-GCM encryption key. Unique per session.

03
AES-256-GCM Encryption

Every chunk is encrypted with a random 12-byte IV and authenticated with a sequence number to prevent replay attacks.

04
DTLS (WebRTC Layer)

WebRTC DataChannels enforce DTLS 1.2 automatically — a second encryption layer at the transport level.
